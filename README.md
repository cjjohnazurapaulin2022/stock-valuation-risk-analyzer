### 📋 Project Overview

This interactive Jupyter Notebook upgrades my AMA 106 Activity 1 case study into a **dynamic valuation and risk analysis tool**.  

Users can adjust any input (growth rates, risk-free rate, beta, economic states, risk aversion, etc.) and instantly see updated results — demonstrating real-world application of financial economics theory.

### ✨ Key Features

- Live market data from Yahoo Finance (NVDA or any ticker)
- Full Discounted Cash Flow (DCF) valuation with Terminal Value
- States of Nature & Payoff Vector analysis
- Expected Return, Variance, and Risk Premium calculation
- Expected Utility Theory for Risk-Neutral vs Risk-Averse investors
- Sensitivity analysis via interactive sliders
- One-click "Calculate / Update Results" button

### 🛠 How to Use

1. Open `NVDA_Interactive_Valuation_Analyzer.ipynb` in Jupyter Notebook
2. Adjust any sliders or inputs
3. Click **"Calculate / Update Results"**
4. Results update instantly
